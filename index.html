DOM

With the HTML DOM, Javascript can access and change all the elements of the HTML documents.

Document Object Model

It defines a standard accessing document

The Document Object Model is a platform and language-neutral for all programs and scripts dynamically accessing and updating the changes, structure and style of the document.

  • DOM is created by the browser when a web page is loaded.
  • In graphical form, it looks like a tree of elements or nodes
  • Programatically, we can use JS to read or change the DOM

Interacting with the DOM

  • Change/Remove HTML elements in the DOM/on the page
  • change & add CSS styles to elements
  • read/change element attributes (href, src, alt, custom)
  • create HTML elements and insert them into the DOM/ the page
  • attach event listeners to elements(click, keypress, submit)

Finding HTML elements

  • Finding HTML element by ID
  • Finding HTML element by Tag name
  • Finding HTNL element by class name
  • Finding HTML element by CSS selector
  • p p.example .example a[target] div>p h2, div, span #demo

  • Finding HTML elemnts by HTML object collections

  • all head, title, body, images, forms links, doctype, url, baseurl, domain
